## ğŸ—ï¸ **How It Works**

<div align="center">

```mermaid
graph TD
    A[User asks question in chat] --> B[Frontend sends to API]
    B --> C[Translate to English if needed]
    C --> D[Generate embedding vector]
    D --> E[Search Pinecone for relevant PDF chunks]
    E --> F[Build context from top 3 matches]
    F --> G[Send to Groq Llama 3.3 70B]
    G --> H[AI generates answer using PDFs]
    H --> I[Translate back to user's language]
    I --> J[Stream response to user]
    
    style A fill:#667eea
    style J fill:#764ba2
    style G fill:#ff6600
```

</div>

### **RAG Pipeline Explained:**

1. **User Question** â†’ "What is sexual harassment under POSH?"
2. **Embedding** â†’ Convert to vector representation
3. **Pinecone Search** â†’ Find similar PDF chunks in database
4. **Context Building** â†’ Combine top 3 relevant sections
5. **AI Prompt** â†’ Send context + question to Llama 3.3 70B
6. **AI Response** â†’ Generate accurate answer based on YOUR PDFs
7. **Translation** â†’ Convert to user's language
8. **Streaming** â†’ Send response in real-time

**Result:** Accurate, fast, contextual answers in any language! ğŸ¯
